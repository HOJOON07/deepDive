## 내부 슬롯과 내부 메서드

> [[]]로 감싼 것들

- [[...내부 슬롯, 내부 메서드]]
- 내부 슬롯과 내부 메서드는 원칙적으로는 개발자가 접근 불가능하다

```
[[Prototype]] => __proto__
```

### 프로토타입

- 어떤 객체의 상위 객체의 역할을 한다.
- 프로토타입은 하위(자식)에게 상속한다.
- 프로퍼티 또는 메서드가 없다면 프로토타입 체인을 따라 프로토타입의 프로퍼티나 메서드를 차레대로 검색한다.
